訪問看護業務フロー

利用開始から終了までの流れ

基本方針

  • 利用者が住み慣れた地域・自宅で安心して生活できるよう、医療・介護両面から支援する
  • 医療保険の訪問看護：主治医の指示書に基づき、医師の指示を遵守して行う
  • 介護保険の訪問看護：ケアプランに基づき、利用者の生活状況と医師の指示を踏まえて行う
  • 医師・ケアマネジャー・介護職・家族と連携し、生活の質（QOL）を維持・向上させる
  • 記録・書類業務を適切に行い、法令遵守と情報共有を徹底する

主な業務内容

区分内容
訪問看護業務バイタルサイン測定・病状観察 / 医師の指示に基づく処置（点滴・吸引・創傷処置等）/ 薬剤管理（残薬確認・服薬指導）/ ターミナルケア（疼痛コントロール・看取り支援）/ 精神的ケア・ご家族への支援
書類業務訪問看護計画書（指示書発行時・ケアプラン変更時・状態変化時）/ 訪問看護報告書（月次・翌月3日まで）/ 訪問記録（訪問ごと・原則訪問日に記載）

業務フローの概要

フェーズ内容
利用開始準備依頼受付 → 訪問看護指示書の取得 → 初回訪問前カンファレンス（利用者情報確認・他職種と情報共有）
サービス導入初回訪問（契約・計画書の説明と同意・状態把握・生活環境確認） → 記録作成
定期訪問（継続支援）医師の指示に基づく処置 → ケアプランに沿った支援 → 家族支援 → 記録・報告（状態変化時は医師・ケアマネへ速やかに報告）
状態変化・緊急対応容体急変時：医師へ連絡 → 必要に応じ救急要請 → ケアマネ・家族へ報告 → 管理者へ連絡 → 記録・再カンファレンス
サービス終了病状改善・入院・施設入所・看取り・利用者希望による終了。関連書類の整理・保管。

情報共有・連携先

連携先内容
主治医状態変化・処置内容の報告、報告書提出
訪問看護管理者状態変化・その他報告事項・利用者家族からの要望・苦情内容
ケアマネジャー計画書・報告書で情報共有。緊急時は随時連絡
多職種定期カンファレンス等での情報交換

書類管理（受け持ち看護師の責任）

計画書・報告書・指示書の期限管理

訪問看護計画書の作成・管理

作成タイミング

  • 訪問看護指示書発行時（初回訪問の場合は初回訪問までに必ず作成する）
  • ケアプラン作成時・更新時
  • 利用者の状態変化があった時
🔏 計画書の代表者氏名欄の横に院長の認印を捺印してもらうこと。捺印のない計画書は正式書類として扱わない。

サイン取得と配布

  • 計画書は2部印刷する
  • 利用者本人または家族（可能であればキーパーソン）より署名をもらう
  • 1部：利用者控えとして利用者または家族へ渡す
  • 1部：法人保管用

保険別の処理方法

保険種別処理方法
医療保険計画書原本を「訪問看護利用者 個別ファイル」へ綴じる
介護保険サイン済み計画書を1部コピーする。原本：個別ファイルへ。コピー：担当ケアマネジャーへ提出する

訪問看護報告書

  • 訪問看護を利用している全利用者について、毎月作成する
  • 作成期限：翌月3日まで
  • 報告書を1部印刷し、「報告書提出ファイル」へ提出する

訪問看護指示書の期限管理

受け持ち看護師は担当する全利用者の指示書の有効期限を把握・管理する責任を負います。
  • 訪問看護指示書期限一覧・訪問看護利用者 個人ファイルにて確認
  • 期限直近の場合は医師へ再度依頼を行う
  • 確認責任は受け持ち看護師が負う

データ管理・USB

  • 訪問看護に関するデータは訪問看護USB内に保存する
  • USBは訪問看護ファイル棚に保管する
🚫 いなだ医院外へのUSBの持ち出しは絶対禁止。紛失・破損・データ不具合があった場合は速やかに訪問看護管理者へ報告すること。

書類の訂正方法（公文書として取り扱う）

🚫 修正テープ・修正液の使用は禁止。記載内容を消去・判読不能にする訂正は禁止。

正しい訂正方法（手書き書類）

  • 誤記部分に二重線を引く（判読可能な状態で）
  • 訂正印または訂正者氏名を記載する
  • 正しい内容を近くに追記する
提出済み書類は無断で差し替えを行わない。訂正・再提出の要否は管理者判断とする。

看護師カンファレンス運用マニュアル

2026年4月13日制定

開催概要

項目内容
開催頻度毎月 1回
開催日時毎月のシフト表にて事前に周知（目安：30分〜1時間以内）
開催場所メディケアビレッジかしわじま 相談室
司会・書記スタッフによる交代制（別紙の「看護師シフト票」を確認）

検討・共有内容

利用者情報共有

  • 新規入所予定者：氏名・入所予定日・利用者背景・ADL等
  • 退所者：氏名・退所日・退所理由
  • 訪問看護開始者：氏名・開始日・開始理由・具体的な訪問内容
  • 訪問看護終了者：氏名・終了日・終了理由

業務改善・連絡事項

  • 日々の気づき・意見・具体的な改善提案
  • 現在進行中の改善取り組みの経過報告
  • その他、全体で周知が必要な事項

議事録の作成と管理

  • 作成期限：カンファレンス終了後3日以内に作成
  • データ保存先：施設看護師用USB（黄色のUSBメモリ）内の原本データをコピーしてPCで編集
  • 印刷は計2枚 — 1枚：統括（稲田崇志）へ提出。1枚：カンファレンスファイルへファイリング
📝 議事録は欠席したスタッフへの重要な情報伝達ツールです。決定事項や経過が誰にでも分かるよう、簡潔かつ正確に記載してください。

業務改善のフロー

  • 協議 — カンファレンス内で意見を出し合い、方向性をまとめる
  • 記載 — 議事録内の「改善提案項目」に記入する際、目印や色付けを付けて分かりやすく記載する
  • 提案 — 書記が議事録を提出する際、統括（稲田崇志）へ直接提案を行う
  • 実施 — 統括の承認を得てから、実際の業務変更・導入を行う
